Editor Tutorial

Hey, you made it! This is the Editor, where you'll be doing most of your writing. Let's check out a few of its features...

Type / to summon the quick menu.

This lets you format your text and create useful content blocks. For example, you can type /h1 to create a heading block, or /quote to create a quote block. View all the options by typing / and navigating with the arrow keys. Press enter to confirm your choice.

Type @ to mention other pages.

Use @-mentions to build a connected knowledgebase. To mention a page that doesnt' exist yet, you can type the desired page name surrounded by double square brackets '[' to quickly create and mention a new sub-page. You can its position in the page hierarchy by dragging and dropping in the sidebar.

Click the menu button in the top right corner of the screen to open up the Page Menu.

Here you can add tabs to this page, auto-detect and create links using Auto-link, export, save and compare versions of this document, and many other things.

You can check out the Editor's many keyboard shortcuts by selecting Help from the quick menu.
